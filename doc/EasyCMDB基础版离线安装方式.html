EasyCMDB基础版离线安装方式
1.上传安装器（cmdb-community-xxxx.tar.gz）到机器上
2. 请将其解压，并mv到/usr/local/ucpro，目录结构如：
[root@DeployTest01 local]# cd ucpro
[root@DeployTest01 ucpro]# pwd
/usr/local/ucpro
[root@DeployTest01 ucpro]# ll
total 36
drwxr-xr-x. 2 root root 4096 Apr 4 17:29 bin
drwxr-xr-x. 2 root root 4096 Apr 4 17:50 log
drwxr-xr-x. 11 root root 4096 Apr 4 17:50 nginx
-rw-r--r--. 1 root root 293 Feb 28 19:57 package.conf.yaml
drwxr-xr-x. 3 root root 4096 Apr 4 17:45 static
drwxr-xr-x. 2 root root 4096 Apr 4 17:29 tools
drwxr-xr-x. 10 root root 4096 Apr 4 17:50 ucpro_service
drwxr-xr-x. 3 root root 4096 Apr 4 17:29 ucpro_web
-rw-r--r--. 1 root root 29 Apr 4 17:29 version.ini
3. 启动安装器./bin/start.sh，并在浏览器登陆http://ip:4200（ip为该机器ip，用chrome浏览器）
4. 点击离线安装，并输入安装的主机IP及主机密码
5. 点击下一步，他会做环境检查。如果有问题，请按提示处理
6. 点击上传部署包（因为我们的这个部署包是已经带有部署包的了，所以可以直接点击下一步）
7. 安装，请耐心等待勿刷新浏览器，一般安装时间为2~3分钟
安装完成后，将会显示默认用户名密码。该密码保存在/root/.easyops文件，也可以在上面查看
8. 服务检查
9. 登录平台
请登录http://ip（该ip为刚才安装机器的ip）
恭喜你，已经可以正常使用了！
10. 卸载，如果需要卸载，请执行如下命令
sh /data/easyops/tmp/uc/tools/easyops_uninstall.sh
